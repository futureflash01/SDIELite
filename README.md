# SDIE Lite

# BACKGROUND INFO
## What is this? 

SDIE Lite is a batch file that downloads and extracts [SDIE](https://github.com/laithayoub71/SDIE) temporarily and deletes all files after command execution. 

This makes SDIE Lite only 12KB instead of 8MB.
